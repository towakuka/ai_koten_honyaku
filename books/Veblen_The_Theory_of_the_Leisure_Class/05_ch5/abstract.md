# ch5.md

第五章「金銭的生活水準」では、人々の支出が単なる身体的快適のためではなく、世間で上品とされる生活水準に合わせるために拡大していくと論じられる。生活水準はいったん習慣化すると下げることが難しく、名誉を示すための浪費でさえ、心の安寧に不可欠なものとなる。各階級は、自分たちより一つ上の階級の消費様式を模倣し、その基準は最終的に有閑階級の慣習へ連なる。競争心と他者との比較は、所得の増加を余暇ではなく、より高い顕示的消費へ向かわせる。そのため産業効率が上がっても労苦は減らず、余剰は体面維持に吸収される。家庭生活の節約、子どもの抑制、学問階級の過重な体面支出なども、この金銭的生活水準の圧力から説明される。
