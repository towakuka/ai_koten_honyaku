# ch2.md

第二章「金銭的競争」では、有閑階級と私有財産が同じ略奪的文化の中から生じたことが論じられる。所有の最初の形は、男性による女性捕虜の所有であり、そこから奴隷や財物の所有へ広がった。富は当初、戦利品として武勇や支配力を示す証拠だったが、産業生活が発達するにつれ、共同体内で他者より優れていることを示す一般的な指標となる。人は単に生存や快適さのために財産を求めるのではなく、他者との比較で尊敬や羨望を得るために蓄積する。いったん富が名声と自尊心の基礎になると、各人は自分の階層の標準以上を求め続けるため、欲望は決して満たされない。ヴェブレンは、近代の経済活動の根底にも、この金銭的優越をめぐる競争心が働いていると見る。
