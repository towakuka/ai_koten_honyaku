# PROLOGO_PARA_FRANCESES.md

「フランス人への序文」は、『大衆の反逆』をフランスの読者に向けて差し出すにあたり、オルテガが本書の読み方を示した文章である。彼はまず、言葉は思想を完全には伝えられず、著者は抽象的な「人類」ではなく具体的な読者に向けて語るべきだと述べる。そのうえで、ヨーロッパは古くから一つの社会を形成してきたが、その強みは単純な統一ではなく、多様性を含んだ統一にあったと論じる。しかし現代には、歴史や内面を失い、権利だけを当然視する「大衆人」が広がり、ヨーロッパを悪しき同質性へ押し流している。したがってこの序文は、本書を単なる政治批評ではなく、現代ヨーロッパの人間類型と文明の危機を問う書物として読むための導入である。
