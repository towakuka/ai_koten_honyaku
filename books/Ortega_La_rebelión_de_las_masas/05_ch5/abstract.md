# ch5.md

第五章「統計上の一事実」では、現代の生がかつてなく豊かな可能性を持ちながら、その可能性を導く原理や計画を欠いていることが論じられる。生とは、与えられた状況の中で何になるかを決断することだが、現代を支配する大衆人は明確な未来像を持たず、その場しのぎに漂流している。オルテガはその背景として、十九世紀以後のヨーロッパ人口の急増を挙げる。自由主義的民主政と技術は、人間の生活力を高め、大衆を歴史の表面へ押し出した。しかし学校制度は技術を教えても、歴史的責任や精神を十分に与えなかった。そのため大衆人は、近代文明の成果を当然視しながら、それを支える原理を理解しない。十九世紀の成果は偉大だが、それ自体が文明を脅かす大衆人を生んだのであり、この人間類型を理解することが現代の最大課題となる。
