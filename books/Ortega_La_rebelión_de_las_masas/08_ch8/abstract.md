# ch8.md

第八章「なぜ大衆は何にでも介入し、しかも暴力によってしか介入しないのか」では、大衆の反逆の核心が、平均的人間の魂の閉塞にあると論じられる。現代の大衆人は、自分の内にある断片的な意見や欲望を「観念」と思い込み、それを真理や規範に照らして吟味しようとしない。彼は聞くこと、議論すること、自分を超える審級に従うことを拒み、自分の俗見をそのまま社会に押しつけようとする。ここに、教養の欠如、すなわち共通の規範へ訴える能力の喪失がある。その結果、大衆の介入は、対話や手続きを経る「間接行動」ではなく、欲望を即座に押し通す「直接行動」、つまり暴力へ向かう。オルテガは、自由主義を敵や少数者と共に生きる高度な文明の形式と見なし、それを失う大衆社会を野蛮化の兆候として捉えている。
