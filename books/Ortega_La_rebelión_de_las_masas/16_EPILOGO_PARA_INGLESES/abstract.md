# EPILOGO_PARA_INGLESES.md

「イギリス人へのエピローグ」では、オルテガがイギリスの歴史的役割と、ヨーロッパ諸国間の相互不理解を論じている。彼は、イギリスを単に個々人の性格ではなく、独自の社会的・集合的なあり方を作り上げた国民として評価する。とくに危機の中で身振り少なく自己修正する力に、ヨーロッパにとっての重要な使命を見る。一方で、イギリスを含む諸国民は互いの生活様式を理解せず、強国は弱国を安易に見下すようになった。十八世紀のコスモポリタニズムやロマン主義が他国の差異に好奇心と敬意を向けたのに対し、現代は差異を軽蔑へ変えている。スペイン内戦をめぐる誤解や、イギリスの平和主義・軍縮への批判も、この相互無知の問題に結びつく。エピローグは、ヨーロッパ再建のためには、各国が互いの異質性を理解し、責任ある歴史的役割を回復する必要があると訴えている。
